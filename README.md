# 个人健身记录 PWA

这是一个手机优先的个人健身记录网站。正式发布到云端后，iPhone 和安卓手机都可以用浏览器打开，也可以添加到手机桌面当作小程序一样使用。

## 现在能记录什么

- 力量训练：按日期记录胸部、肩部、背部、手臂、腹部。
- 每个动作：记录重量、次数、组数和添加时间。
- 爬坡：单独记录时长和备注。
- 身体数据：记录体重、手臂围、腰围。
- 统计图：按日期范围查看身体数据变化、动作重量/次数/总训练量变化。
- 导出：支持 CSV 表格和 JSON 完整备份。

## 数据放在哪里

数据现在先保存在手机浏览器本地，也就是 IndexedDB。

- 优点：不用登录，打开就能记。
- 注意：不同手机之间不会自动同步。
- 换手机或清理浏览器前，请先在“导出”里保存 JSON 备份。

## 导出文件是什么

- CSV：表格文件，适合用 Excel、WPS、Numbers 打开，方便查看明细。
- JSON：完整备份文件，适合以后导回这个网站，换手机或恢复数据时用。

## 本地预览

电脑开着时，可以用本地地址预览：

```powershell
npm.cmd run dev
```

这只是开发预览。电脑关机后，本地地址就打不开。

## 正式上线

要做到“电脑关机也能打开”，需要把网站发布到云端托管平台，例如 Cloudflare Pages、Vercel 或 Netlify。

发布时一般填写：

- Build command: `npm run build`
- Output directory: `dist`

发布成功后会得到一个 `https://...` 的网址。这个网址只要手机有网就能打开，不依赖电脑开机。

## 安装到手机桌面

iPhone：用 Safari 打开正式网址，点分享按钮，选择“添加到主屏幕”。

安卓：用 Chrome 打开正式网址，点菜单，选择“添加到主屏幕”或“安装应用”。
